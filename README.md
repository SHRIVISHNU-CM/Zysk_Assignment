To get started ,

**Clone the Repository**,

```bash
  git clone https://github.com/Sheinode-Infotech/SIW1.git
```
**install material UI**,

```bash
  npm i @mui/icons-material @mui/material @emotion/styled @emotion/react
```

**Run**,

```bash
  npm run dev
```
